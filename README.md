## Q3: Explain the difference between any, unknown, and never types in TypeScript.
any: কোনও ধরণের টাইপ চেকিং নেই | <br>
unknown: ব্যবহার করার আগে টাইপ চেক করতে হবে |  <br>
never: এমন ফাংশন যা কখনও ফলাফল ফেরত দেয় না | <br>

## Q5: Provide an example of using union and intersection types in TypeScript.
union type: টাইপ একাধিক ধরণের হতে পারে| <br>
intersection type: অবশ্যই উভয় টাইপ অন্তর্ভুক্ত করতে হবে।
